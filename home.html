The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

HTML uses "elements" to let the browser know how a webpage is made of. Elements are shown as "tags" in the code, written with angle brackets: . Tags usually come in pairs: an opening tag defines the start of a block of content and a closing tag defines the end of that block of content. There are many different kinds of tags, and each one has a different purpose. See Basic HTML Tagsbelow for tag examples. Some tags only work in certain browsers
